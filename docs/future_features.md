# グループTODO - 今後の実装予定機能

## ドキュメント概要

- **作成日**: 2025-11-06
- **対象**: 今後実装を検討している機能
- **目的**: 将来的な機能拡張の計画を記録する

---

## 1. 通知機能（未実装）

### 1.1 通知設定

**現状**:
- データベースには通知設定フィールド（notification_deadline, notification_new_todo, notification_assigned）が存在
- しかし画面実装は存在しない

**実装予定**:
- 設定画面に通知設定セクションを追加
- 期限通知、新規TODO通知、担当TODO通知のON/OFF切り替え

### 1.2 プッシュ通知

**現状**:
- プッシュ通知は実装されていない
- グループ招待も画面内のセクション表示のみ

**実装予定**:
- 招待された時のプッシュ通知
- 期限が近いTODOの通知
- 新規TODOが追加された通知
- 担当に設定された通知
